# Love Matcha Sales v1.3.6

อัปเดตนี้แก้ตามคำขอเรื่องพื้นที่กรอกข้อมูลบนมือถือและลดข้อความยาวในหน้าค่าตอบแทน

## สิ่งที่แก้ใน v1.3.6

1. **หน้ายอดขาย**
   - ยกเลิกการล็อก/ฟิกแถบ `เดือน / วันที่ / สาขา / หยุดร้าน` ตอนเลื่อนหน้าจอ
   - จัดแถบวันที่/สาขาใหม่ให้เตี้ยลงและกินพื้นที่น้อยลง
   - เพิ่มความกว้างของฟอร์มบนจอกว้าง เพื่อให้ช่องกรอกวางได้หลายคอลัมน์และกรอกง่ายขึ้น
   - บนมือถือจัดช่องสำคัญเป็น 2 คอลัมน์เมื่อจอกว้างพอ และปรับเป็น 1 คอลัมน์เฉพาะจอแคบมาก

2. **หน้าค่าตอบแทน**
   - ตัดคำอธิบายยาว ๆ ออกจากหน้าหลัก
   - เหลือเฉพาะช่องเลือกเดือน ปุ่มคำนวณใหม่ ตารางค่าตอบแทน และส่วนตั้งค่าที่จำเป็น
   - ลด padding/margin ของแผงตั้งค่า เพื่อให้เห็นตารางและช่องกรอกมากขึ้น

3. **เวอร์ชัน / Cache**
   - อัปเดตชื่อแอปเป็น `v1.3.6`
   - เปลี่ยน cache service worker เป็น v1.3.6 เพื่อให้มือถือโหลดไฟล์ใหม่หลังอัปโหลดขึ้น GitHub

## วิธีอัปเดต

อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ทับของเดิมบน GitHub Pages แล้วเปิดเว็บใหม่อีกครั้ง

ถ้ามือถือยังเห็นหน้าเก่า ให้ปิดแอปจาก recent apps แล้วเปิดใหม่ หรือเข้า Settings ของ browser แล้วล้าง cache เฉพาะเว็บนี้
