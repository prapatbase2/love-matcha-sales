# Love Matcha Sales v1.3.9

## สิ่งที่แก้ใน v1.3.9

1. หน้ายอดขาย
   - ตัดสาขา “ออกบูธ” ออกจากระบบเตือนยอดขาย/หยุดร้านย้อนหลังของเมื่อวาน
   - เหตุผล: ออกบูธเป็นสาขาพิเศษ เปิดเฉพาะบางวันหรือบางเดือน จึงไม่จำเป็นต้องลง “หยุดร้าน” ให้ครบทุกวัน
   - สาขาประจำอื่น ๆ ยังเตือนเหมือนเดิม เพื่อกันลืมลงยอดขายหรือกดหยุดร้าน

2. ความเร็วและความลื่นในการใช้งาน
   - เพิ่ม cache ชั่วคราวสำหรับข้อมูลยอดขายรายเดือน/ช่วงวันที่, เช็คชื่อ, ค่าตอบแทน และรายจ่าย เพื่อไม่ต้องดึงข้อมูลซ้ำทันทีเวลาเปลี่ยนหน้า/กดโหลดใหม่
   - หน้ายอดขายลดการคำนวณซ้ำทุกครั้งที่พิมพ์ ทำให้กรอกตัวเลขในมือถือจอแคบลื่นขึ้น
   - หน้า “เงินสดเปิดกะ / แก้วคงเหลือวันก่อน” ใช้ข้อมูลที่โหลดไว้ชั่วคราว ลดการดึงข้อมูลซ้ำเมื่อสลับวันที่หรือสาขา
   - เมื่อมีการบันทึกยอดขาย/เช็คชื่อ/ค่าตอบแทน/รายจ่าย ระบบจะล้าง cache ที่เกี่ยวข้องให้อัตโนมัติ เพื่อไม่ให้ข้อมูลเก่าเกินไป
