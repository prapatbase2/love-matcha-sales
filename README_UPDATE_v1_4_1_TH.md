# Love Matcha Sales v1.4.1

## แก้ด่วนจาก v1.4.0

- แก้สาเหตุเปิดโปรแกรมแล้วขึ้น “โหลดระบบไม่สำเร็จ” จาก JavaScript syntax error ใน `app.js`
- ลบโค้ดตารางยอดขายที่ซ้ำ/หลุด ทำให้ browser parse ไม่ผ่าน
- อัปเดตเลขเวอร์ชันเป็น v1.4.1 และเปลี่ยน Service Worker cache ใหม่
- ลดไฟล์ที่โหลดตอนเปิดแอป: Chart.js และ html2pdf จะโหลดเฉพาะตอนต้องใช้จริง
- เพิ่ม preconnect ไป Firebase เพื่อให้เริ่มโหลดเร็วขึ้น
- เพิ่มไฟล์ `เปิดทดสอบในเครื่อง.bat` สำหรับเปิดผ่าน local server ไม่ใช่ file://

## วิธีอัปโหลด

1. แตก ZIP
2. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์ `love_matcha_sales_app_v1_4_1` ทับของเดิมใน GitHub
3. รอ GitHub Pages deploy เสร็จ
4. เปิดเว็บแล้วกด Refresh 1-2 ครั้ง หรือปิด/เปิดแอปใหม่ เพื่อให้ Service Worker รับ v1.4.1

## ถ้ายังเห็น v1.4.0

ให้ล้าง cache ของเว็บหรือกด Reload หลายครั้ง เพราะมือถืออาจยังถือ Service Worker เก่าอยู่
