# Love Matcha Sales v1.3.6

## สิ่งที่แก้ใน v1.3.6

1. หน้ารายเดือน
   - ลบข้อความคำอธิบายใต้หัวข้อ “ตารางรายวันแยกตามสาขา” ออกแล้ว
   - ตารางรายวันแยกตามสาขายังคงเรียงจากต้นเดือนไปปลายเดือนตามเดิม แต่ไม่แสดงคำอธิบายยืดยาวบนหน้า

2. หน้ารายเดือนบนมือถือ
   - แก้ปัญหากด “แสดงรายละเอียด” แล้วรายละเอียดไปดันความกว้างของตารางหลัก
   - ตารางหลักคงความกว้างเดิม และรายละเอียดเลื่อนซ้าย-ขวาอยู่ภายในช่องรายละเอียดเอง
   - ยังแสดงตัวอักษร ตัวเลข และข้อมูลครบถ้วน ไม่ตัดข้อมูลสำคัญ

3. เวอร์ชัน
   - อัปเดตเลขเวอร์ชันเป็น v1.3.6
   - เปลี่ยน cache ของ Service Worker เพื่อให้มือถือโหลดไฟล์ใหม่หลังอัปโหลด

## วิธีอัปเดต

อัปโหลดไฟล์ทั้งหมดใน ZIP นี้ทับไฟล์เดิมบน GitHub แล้วรอ GitHub Pages deploy เสร็จ จากนั้นเปิดเว็บแล้วกด refresh 1-2 ครั้ง หรือปิด/เปิดแอปใหม่บนมือถือ เพื่อให้ Service Worker โหลด v1.3.6
