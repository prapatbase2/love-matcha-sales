# Love Matcha Sales v1.3.6

## สิ่งที่แก้ใน v1.3.6

1. หน้ายอดขาย
   - เพิ่มความเข้มของสีพื้นหลังแต่ละกลุ่มให้มองเห็นชัดขึ้น แต่ยังไม่เข้มจนลายตา
   - ลดความสูงช่องกรอกและ padding ของ input/select/textarea
   - บนมือถือจอแคบ ปรับช่องข้อมูลยอดขาย/เงินสด/แก้ว/OT ให้แสดง 2 ช่องต่อบรรทัดเพื่อลดความยาวหน้า
   - คงปุ่มบันทึกชั่วคราวและบันทึกจริงไว้ 2 ช่องคู่กันบนมือถือ

2. หน้าสรุปรายเดือน > แสดงรายละเอียด
   - เปลี่ยนรายละเอียดเป็นตารางคู่ซ้าย/ขวา: “ยอดขาย” และ “เงินสด/รายจ่าย”
   - รายการรายจ่ายอื่นแสดงแบบ 2 ช่องซ้าย/ขวา ลดการเลื่อนยาวลงล่าง
   - ถ้ากว้างเกินจอมือถือ สามารถเลื่อนซ้าย/ขวาได้ โดยตัวหนังสือและตัวเลขยังเห็นครบ
   - ใช้สีพื้นหลังให้สอดคล้องกับหน้ายอดขาย

## วิธีอัปเดต

อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้น GitHub Pages ทับของเดิม แล้วเปิดเว็บใหม่ หากมือถือยังเห็นเวอร์ชันเก่า ให้ปิดแอป/เบราว์เซอร์แล้วเปิดใหม่ หรือกดรีเฟรช 1-2 ครั้ง เพราะ v1.3.6 เปลี่ยน service worker cache แล้ว
