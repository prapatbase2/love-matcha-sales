# Love Matcha Sales v1.3.6

## สิ่งที่แก้ใน v1.3.6

1. หน้ารายเดือน
   - ลบคำอธิบายใต้หัวข้อ “ตารางรายวันแยกตามสาขา” ออกแล้ว
   - ตารางรายวันแยกตามสาขายังเรียงจากวันที่ 1 ไปปลายเดือนเหมือนเดิม แต่ไม่แสดงข้อความอธิบายยาว

2. ปุ่ม “แสดงรายละเอียด” ในมือถือ
   - แก้ไม่ให้รายละเอียดไปดันความกว้างของตารางหลัก
   - ตารางหลักล็อกความกว้างคงที่ ใช้เลื่อนซ้าย-ขวาตามเดิม
   - รายละเอียดด้านในมีพื้นที่เลื่อนซ้าย-ขวาแยกของตัวเอง ทำให้ตัวอักษร/ตัวเลขครบ ไม่ซ้อนกัน และไม่ทำให้ช่องหลักยืด

3. อัปเดตเวอร์ชัน
   - อัปเดตเลขเวอร์ชันเป็น v1.3.6
   - เปลี่ยน cache ของ PWA เพื่อให้มือถือโหลดไฟล์ใหม่หลังอัปโหลดทับ

## วิธีอัปเดต

อัปโหลดไฟล์ทั้งหมดใน ZIP นี้ทับไฟล์เดิมบน GitHub แล้วรอ GitHub Pages deploy เสร็จ จากนั้นเปิดเว็บแล้วกด refresh 1-2 ครั้ง หรือปิด/เปิดแอปใหม่บนมือถือ เพื่อให้ Service Worker โหลด v1.3.6
