# Love Matcha Sales v1.3.7

## สิ่งที่แก้ใน v1.3.7

1. หน้ารายเดือน: ลบคำอธิบายใต้หัวข้อ “ตารางรายวันแยกตามสาขา” ออกแล้ว
   - ไม่มีข้อความ “เรียงจากต้นเดือน...” หรือคำอธิบายยาวใต้หัวตาราง

2. หน้ารายเดือนบนมือถือ: แก้ปัญหากด “แสดงรายละเอียด” แล้วตารางถูกดันให้กว้างขึ้น
   - บังคับตารางรายวันให้ใช้ขนาดคงที่เหมาะกับมือถือ
   - รายละเอียดของวันนั้นอยู่ในพื้นที่ของตารางเดิม ไม่ทำให้ตารางยืดกว้างเพิ่ม
   - ข้อมูลตัวอักษรและตัวเลขยังแสดงครบ อ่านได้ ไม่ซ้อนกัน
   - ตารางยอดขาย / เงินสด-รายจ่าย / รายจ่ายอื่น ๆ จัดให้ย่อขนาดและตัดบรรทัดอย่างเหมาะสมบนมือถือ

3. อัปเดตเลขเวอร์ชันเป็น v1.3.7
   - `index.html`
   - `app.js`
   - `sw.js` cache ใหม่ เพื่อให้มือถือโหลดไฟล์ใหม่แทน cache เก่า
   - เอกสารประกอบ

## วิธีอัปเดต

อัปโหลดไฟล์ทั้งหมดใน ZIP นี้ทับไฟล์เดิมบน GitHub Pages ให้ครบทุกไฟล์ โดยเฉพาะ:

- `index.html`
- `app.js`
- `style.css`
- `sw.js`
- `manifest.webmanifest`
- โฟลเดอร์ `icons`

หลังอัปโหลดเสร็จ ให้เปิดเว็บแล้วกด refresh 1-2 ครั้ง หรือปิด/เปิดแอปใหม่บนมือถือ เพื่อให้ Service Worker โหลด v1.3.7
