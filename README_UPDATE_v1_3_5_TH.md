# Love Matcha Sales v1.3.5

อัปเดตจาก v1.3.4 ตามคำขอ:

1. หน้ายอดขาย
- ปรับสีพื้นหลังแต่ละกลุ่มให้เข้มขึ้นกว่าของเดิม แต่ยังเป็นโทนอ่อนอ่านง่าย
- บนมือถือจอแคบ ปรับช่องกรอกหลายส่วนเป็น 2 ช่องต่อบรรทัด เพื่อให้หน้าสั้นลง
- ลด padding ช่องกรอกและปุ่มบางจุดให้เหมาะกับตัวเลขหลักไม่เกินหลักแสน
- รายจ่ายอื่นและ OT ทำขนมบนมือถือจัดเป็น 2 ช่องต่อบรรทัด พร้อมให้ช่องหมายเหตุกินเต็มบรรทัดเพื่ออ่านครบ

2. หน้ารายเดือน / ตารางรายวัน
- เมื่อกด “แสดงรายละเอียด” รายละเอียดยอดขายและเงินสด/รายจ่ายวางคู่ซ้าย-ขวา
- รายการรายจ่ายอื่นจัดเป็น 2 ช่องต่อบรรทัดเพื่อลดความยาวหน้า
- ใส่สีพื้นหลังรายละเอียดให้ตรงกับกลุ่มสีในหน้ายอดขาย
- ปรับรายละเอียดบนมือถือให้ตัวอักษรและตัวเลขยังเห็นครบ ไม่ล้นกรอบง่าย

3. เวอร์ชัน
- อัปเดตเป็น v1.3.5 ใน app.js, index.html และ service worker cache

วิธีใช้งาน: อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ทับของเดิมบน GitHub Pages แล้วเปิดเว็บใหม่ หากมือถือยังเห็นเวอร์ชันเก่า ให้ปิดแอป/เบราว์เซอร์แล้วเปิดใหม่ หรือเคลียร์แคช/PWA หนึ่งครั้ง
