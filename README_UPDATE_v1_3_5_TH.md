# Love Matcha Sales v1.3.5

อัปเดตตามคำขอเรื่องหน้าจอมือถือจอแคบและหน้ารายละเอียดรายเดือน

## สิ่งที่แก้ใน v1.3.5

1. หน้ายอดขาย
   - ปรับสีพื้นหลังแต่ละหมวดให้เข้มขึ้นกว่ารุ่น v1.3.4 แต่ยังไม่เข้มจนรบกวนสายตา
   - ลด padding / ความสูงของช่องกรอกบนมือถือ
   - บนมือถือจอแคบ ช่องกรอกในหมวดหลักแสดง 2 ช่องต่อบรรทัด เพื่อให้หน้าไม่ยาวเกินไป
   - รายจ่ายอื่น ๆ แสดงชื่อรายการและจำนวนเงินคู่กันซ้าย/ขวา และให้ช่องหมายเหตุเต็มบรรทัด
   - ปุ่มบันทึกชั่วคราวและบันทึกจริงยังอยู่ด้านล่าง ใช้งานเหมือนเดิม

2. หน้ารายเดือน > ตารางรายวัน > แสดงรายละเอียด
   - เปลี่ยนรายละเอียดให้เป็นตารางคู่: “ตารางยอดขาย” และ “ตารางเงินสด / รายจ่าย” วางซ้ายขวา
   - รายการรายจ่ายอื่นแสดงแบบ 2 ช่องต่อบรรทัดเมื่อพื้นที่พอ
   - ใช้สีพื้นหลังให้ตรงกับหมวดในหน้ายอดขาย เพื่ออ่านง่ายและไม่งง

3. เวอร์ชัน
   - อัปเดตเลขเวอร์ชันเป็น v1.3.5
   - เปลี่ยน cache ของ PWA แล้ว เพื่อช่วยให้มือถือโหลดไฟล์ใหม่หลังอัปโหลดทับ

## วิธีอัปเดต

ให้อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ทับไฟล์เดิมบน GitHub Pages / โฮสต์เดิม แล้วเปิดเว็บใหม่ หากมือถือยังเห็นหน้าเก่า ให้ปิดแอปจาก recent apps แล้วเปิดใหม่ หรือ refresh หน้าเว็บ 1-2 ครั้ง
