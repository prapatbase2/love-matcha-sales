# Love Matcha Sales v1.3.5

## สิ่งที่แก้ใน v1.3.5

1. หน้ายอดขาย
   - ปรับสีพื้นหลังของแต่ละกลุ่มให้เข้มและเห็นชัดขึ้น แต่ยังไม่มืดเกินไป
   - ปรับช่องกรอกให้แคบและกระชับขึ้น
   - บนมือถือจอแคบ กลุ่มช่องตัวเลขจัดเป็น 2 ช่องต่อบรรทัดเมื่อพื้นที่พอ เพื่อให้หน้าสั้นลงและเลื่อนน้อยลง
   - รายจ่ายอื่น ๆ จัดให้กรอกแบบ 2 ช่องต่อแถว และช่องหมายเหตุเต็มแถว ลดความยาวหน้า

2. หน้ารายเดือน / รายละเอียดยอดขายรายวัน
   - เมื่อกด “แสดงรายละเอียด” จัดรายละเอียดเป็นตารางคู่แนวนอน
   - ตารางยอดขายอยู่ซ้าย ตารางเงินสด/รายจ่ายอยู่ขวา และรายการรายจ่ายอื่นอยู่ถัดไป
   - รองรับมือถือโดยให้เลื่อนซ้าย-ขวาได้ ตัวอักษรและตัวเลขยังแสดงครบ
   - สีพื้นหลังของรายละเอียดรายเดือนปรับให้เข้ากับหน้ายอดขาย

3. ระบบ PWA / Cache
   - อัปเดตเลขเวอร์ชันเป็น v1.3.5
   - เปลี่ยนชื่อ cache ของ Service Worker เพื่อให้มือถือโหลดไฟล์ใหม่หลังอัปโหลดขึ้น GitHub

## วิธีอัปเดต

อัปโหลดไฟล์ทั้งหมดใน ZIP นี้ทับไฟล์เดิมบน GitHub แล้วรอ GitHub Pages deploy เสร็จ จากนั้นเปิดเว็บแล้วกด refresh 1-2 ครั้ง หรือปิด/เปิดแอปใหม่บนมือถือ เพื่อให้ Service Worker โหลด v1.3.5
