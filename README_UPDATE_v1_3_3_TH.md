# Love Matcha Sales v1.3.3

## สิ่งที่แก้ในเวอร์ชันนี้

1. หน้าเบิกเงิน
   - ผู้ใช้ทั่วไปลบรายการเบิกเงินของตัวเองได้
   - เจ้าของและผู้จัดการลบรายการเบิกเงินของทุกคนได้
   - มีปุ่มลบในตารางรายการเบิกเงิน พร้อมยืนยันก่อนลบ

2. หน้ายอดขาย
   - ปรับ UI ให้ไม่ลายตาบนมือถือจอแคบ
   - แยกกลุ่มด้วยพื้นหลังสีจาง:
     - พนักงานในกะ
     - ข้อมูลยอดขาย
     - เงินสด / รายจ่าย
     - ข้อมูลแก้ว
     - OT ทำขนม
     - หมายเหตุ

3. หน้าสรุปรายเดือน
   - ตารางรายวันเรียงจากวันที่ 1 ลงไปปลายเดือน
   - แยกตารางแต่ละสาขาออกจากกัน
   - ตารางหลักแสดงเฉพาะ:
     - วันที่
     - ชื่อพนักงานในกะ
     - รายได้รวมทั้งหมด
     - รายจ่ายรวมทั้งหมด
     - เงินสดให้เจ้าของ
     - ปุ่มแสดงรายละเอียด
   - วันหยุดร้านแสดงคำว่า “หยุดร้าน” และไม่แสดงยอดเงิน
   - วันที่เปิดร้านกด “แสดงรายละเอียด” และ “ปิด/ซ่อนรายละเอียด” ได้

4. หน้าค่าตอบแทน
   - เพิ่มช่องกำหนดค่าตอบแทนพนักงานรายวันหลังตารางค่าตอบแทนรายวัน และก่อนตั้งค่าขนม
   - เจ้าของ/ผู้จัดการแก้ได้
   - กำหนดได้ 2 ค่า:
     - ทำงานทั้งวัน บาท/วัน
     - รายชั่วโมง บาท/ชั่วโมง
   - ถ้าทำงานครึ่งชั่วโมง ระบบคิดครึ่งหนึ่งของรายชั่วโมงอัตโนมัติ
   - เงินเดือนพนักงานทั่วไป ถ้าเดือนนี้ยังไม่เคยบันทึก จะลอกจากเดือนก่อนหน้าให้อัตโนมัติ แต่ยังแก้ได้

5. ความเร็วตอน Save / ส่งยอด
   - Auto Backup แบบ “สำรองเมื่อมีการทำรายการ” จะทำงานเบื้องหลัง ไม่รอให้ Backup เสร็จก่อน
   - ช่วยลดอาการกด Save แล้วหน่วงหรือกระตุก

6. Backup / Restore
   - ยกเลิก Backup แบบ Google Sheet / Excel
   - เหลือเฉพาะไฟล์ `.json`
   - Restore รองรับเฉพาะ `.json`
   - ตัดการโหลดไลบรารี Excel ออกจากหน้าเว็บและ Service Worker แล้ว

## วิธีลงไฟล์

1. แตก ZIP นี้
2. อัปโหลดไฟล์ทั้งหมดทับของเดิมใน GitHub Pages repository
3. ต้องอัปโหลดให้ครบ โดยเฉพาะ:
   - `index.html`
   - `app.js`
   - `style.css`
   - `sw.js`
   - `manifest.webmanifest`
   - โฟลเดอร์ `icons`
   - `apps-script-backup.gs`
4. เปิดเว็บแอปผ่าน GitHub Pages
5. ถ้าในมือถือยังเห็นหน้าเดิม ให้ปิดแอปออกจาก recent apps แล้วเปิดใหม่ หรือกด refresh 2-3 ครั้ง เพราะ PWA มี cache

## วิธีอัปเดต Google Apps Script Backup

ต้องทำใหม่ เพราะ v1.3.3 ยกเลิก Google Sheet/Excel และให้สร้าง JSON อย่างเดียว

1. เปิด Google Apps Script ตัวเดิมที่ใช้ Web App URL อยู่
2. เปิดไฟล์ `apps-script-backup.gs` จาก ZIP นี้
3. คัดลอกโค้ดทั้งหมดไปวางทับ `Code.gs` ทั้งไฟล์
4. กด Save
5. ไปที่ `Deploy` > `Manage deployments`
6. กดรูปดินสอของ Web App เดิม
7. ช่อง `Version` เลือก `New version`
8. ตั้งค่า:
   - Execute as: Me
   - Who has access: Anyone
9. กด Deploy
10. กลับเข้าแอปหน้า “สำรอง” แล้วกด “ทดสอบสร้างไฟล์ใน Drive”

ผลที่ถูกต้องควรขึ้น:

- `Love Matcha Sales Backup v1.3.3`
- มี `jsonFileName`
- มี `folderUrl`
- ไม่มี `sheetName`

ถ้ายังขึ้น v1.2 หรือยังมี `sheetName` แปลว่ายังไม่ได้ Deploy เป็น New version ของ Apps Script ตัวเดิม
