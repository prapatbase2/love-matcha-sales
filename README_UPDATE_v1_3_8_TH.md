# Love Matcha Sales v1.3.8

## สิ่งที่เพิ่ม/แก้ใน v1.3.8

1. Dashboard ใหม่
   - เลือกช่วงวันที่ย้อนหลังได้
   - เลือกแสดงรวมทุกสาขา หรือเลือกเฉพาะบางสาขาได้
   - แสดงรายได้รวม, รายจ่ายรวมจากหน้ายอดขาย, รายจ่ายเจ้าของลง, เอาเงินสดให้เจ้าของ, จำนวนแก้วที่ใช้ และรายได้หลังหักค่าใช้จ่าย
   - รายการล่าสุดแสดงวันที่ / สาขา / สถานะ / รายได้รวม / รายจ่ายรวม / เอาเงินสดให้เจ้าของ / เงินสดขาดเกิน / หมายเหตุ

2. เพิ่มหน้า “ลงรายจ่าย”
   - แสดงเฉพาะเจ้าของและผู้จัดการ
   - รองรับรายจ่ายประจำเพิ่มเติมที่จดจำไปเดือนถัด ๆ ไป
   - รองรับรายจ่ายอื่น ๆ แบบเลือกวันที่และเลือกได้ว่าเป็นรวมทุกสาขาหรือเฉพาะสาขา
   - รายจ่ายรวมทุกสาขา และค่าตอบแทนของคนที่อยู่หลายสาขา จะกระจายตามสัดส่วนยอดขายของแต่ละสาขาในเดือนนั้น
   - ดึงค่าตอบแทน/เงินเดือนจากหน้าค่าตอบแทนเป็นรายจ่ายเจ้าของลงอัตโนมัติ

3. หน้ายอดขาย
   - ถ้าเมื่อวานยังไม่ได้ลงยอดขายหรือกดหยุดร้านของสาขาที่มีสิทธิ์ จะขึ้นคำเตือนให้ย้อนกลับไปลงข้อมูล

4. หน้าเช็คชื่อ
   - พนักงานที่อยู่สาขาหนองคายมีช่องติ๊กพาร์ทไทม์ Rendo
   - เลือกเวลา Rendo ได้ตั้งแต่ 18:00 ถึง 22:00 ทีละครึ่งชั่วโมง
   - มีตารางลงเวลา Rendo แยกด้านล่าง
   - ตารางเช็คชื่อเรียงวันที่ 1 ไปปลายเดือน
   - ถ้าวันที่ผ่านไปแล้วยังไม่ได้เช็คชื่อ จะขึ้นเตือนวันที่ที่ยังขาด

5. หน้าค่าตอบแทน
   - เพิ่ม OT พาร์ทไทม์ Rendo เฉพาะข้อมูลจากสาขาหนองคาย
   - เจ้าของ/ผู้จัดการตั้งค่าอัตราค่าตอบแทน Rendo บาทต่อชั่วโมงได้
   - OT ทำขนมจะคำนวณตามชื่อขนม/ราคา/% OT ปัจจุบัน หากแก้ชื่อ ราคา หรือเปอร์เซ็นต์ แล้วคำนวณใหม่ ยอดจะเปลี่ยนตาม

6. หน้าระบบ
   - ประวัติระบบไม่แสดงรายการ Backup/สำรองข้อมูลแล้ว
