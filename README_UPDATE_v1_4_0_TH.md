# Love Matcha Sales v1.4.0

อัปเดตหลักในเวอร์ชันนี้

## Dashboard
- เพิ่มตัวกรองช่วงวันที่: เดือนนี้ / เลือกตั้งแต่วันไหนถึงวันไหนย้อนหลังได้
- เลือกดูรวมทุกสาขา หรือเลือกเฉพาะบางสาขาได้
- เพิ่ม KPI: รายได้รวม, รายจ่ายรวมจากหน้ายอดขาย, รายจ่ายเจ้าของลง, เงินสดให้เจ้าของ, จำนวนแก้วที่ใช้รวม, รายได้หลังหักค่าใช้จ่าย
- รายการล่าสุดแสดง: วันที่, สาขา, สถานะ, รายได้รวมทั้งหมด, รายจ่ายรวมทั้งหมด, เงินสดให้เจ้าของ, เงินสดขาด/เกิน, หมายเหตุ

## ยอดขาย
- เพิ่มคำเตือนถ้าเมื่อวานยังไม่ได้ลงยอดขายหรือกดหยุดร้านของสาขาปกติ
- ยกเว้นสาขา “ออกบูธ” ไม่เตือนย้อนหลัง เพราะเป็นสาขาพิเศษที่ไม่ได้เปิดประจำ

## เช็คชื่อ
- ตารางเช็คชื่อเรียงจากวันที่ 1 ไปปลายเดือน
- แจ้งเตือนวันที่ย้อนหลังในเดือนนั้นที่ยังไม่ได้เช็คชื่อ
- พนักงานที่อยู่สาขาหนองคายมีช่องติ๊ก “พาร์ทไทม์ Rendo” และเลือกเวลา 18:00-22:00 ทีละครึ่งชั่วโมง
- เพิ่มตารางลงเวลา พาร์ทไทม์ Rendo ด้านล่างเมื่อมีรายการ

## ค่าตอบแทน
- เพิ่มช่องตั้งค่า “พาร์ทไทม์ Rendo บาท/ชั่วโมง” เฉพาะเจ้าของ/ผู้จัดการ
- เพิ่มคอลัมน์ OT พาร์ทไทม์ Rendo ในตารางค่าตอบแทน
- OT ทำขนมคำนวณจากราคาขาย/% OT ปัจจุบันในหน้าค่าตอบแทน ทำให้เมื่อปรับราคา/% แล้วคำนวณใหม่ตามจริง
- แก้ syntax เดิมในหน้าค่าตอบแทนที่ทำให้บางเครื่องเสี่ยงคำนวณ/โหลดผิดพลาด

## ลงรายจ่าย
- เพิ่มเมนู “ลงรายจ่าย” เฉพาะเจ้าของ/ผู้จัดการ อยู่ถัดจากเช็คชื่อ ก่อนเบิกเงิน
- รองรับรายจ่ายประจำ และรายจ่ายอื่นรายวัน
- เลือกได้ว่าเป็นรายจ่ายรวมทุกสาขา หรือของสาขาใดสาขาหนึ่ง
- รายจ่ายรวมทุกสาขากระจายตาม % ยอดขายของแต่ละสาขาในเดือนนั้น
- ดึงต้นทุนค่าตอบแทนที่บันทึกแล้วมาเป็นรายจ่ายประจำอัตโนมัติ
- รายจ่ายประจำที่เพิ่มเองจะจำไปเดือนถัด ๆ ไป และแก้ไข/ลบได้
- รายจ่ายอื่นแก้ไข/ลบได้

## ระบบ
- หน้าประวัติระบบไม่แสดงรายการสำรองข้อมูลแล้ว
- อัปเดต cache/service worker เป็น v1.4.0 เพื่อให้มือถือโหลดไฟล์ใหม่
